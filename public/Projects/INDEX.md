## 🚀 Guide de Démarrage

### Pour accéder à un projet spécifique :

1. **Projets GitHub** :
   - Allez sur https://github.com/Ahmeday-ui
   - Cherchez le repository souhaité
   - Consultez le `README.md` pour instructions d'installation

2. **Rapports HTML** :
   - Double-cliquez sur le fichier `.html`
   - Ouvrez dans votre navigateur web

3. **Rapport PDF** :
   - Double-cliquez sur le fichier `.pdf`
   - Ouvrez dans votre lecteur PDF

### Technologies principales utilisées

- **Python** : Deep Learning, Data Science, Machine Learning
- **R** : Statistique, Séries Temporelles, Visualisation
- **JavaScript** : Interactivité web (via Shiny/HTML)
- **SQL** (optionnel) : Requêtes sur données structurées
